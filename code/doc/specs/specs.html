AIPS++ Specifications

The initial AIPS++ Consortium User Specifications were written in late 1992. These have been augmented in 1995 by:
  • NFRA Requirements for AIPS++ by J.E. Noordam
  • ATNF Calibration requirements for AIPS++ by M.H. Wieringa
  • VLBI Requirements for AIPS++ by A.J. Beasley
  • BIMA calibration requirements for AIPS++ by P. Teuben
  • Mosaicing requirements for AIPS++ by M. Holdaway, R.J. Sault, and T.J. Cornwell
Please send any comments or questions about AIPS++ to aips2-requests@nrao.edu
