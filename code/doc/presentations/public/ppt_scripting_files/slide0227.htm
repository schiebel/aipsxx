Quanta
nQuanta are values with units contained in a record
nA variety of physical constants and decimal prefixes are available
nQuanta may be combined algebraically:
n
n   - include 'quanta.g’
n   - d:=25      # the diameter of a typical telescope
n   - print A:=dq.quantity(pi*(d^2)/4,’m2’)
n     [value=490.873852, unit=m2]
n   - eff:=0.60    # its aperture efficiency
n   - print k:=dq.constants('k’)    # Boltzmann's constant
n     [value=1.3806578e-23, unit=J/K]
n   - print g:=dq.div( dq.mul(eff,A), dq.mul(2,k))
n     [value=1.06660865e+25, unit=m2/(J/K)]
n   - print g:=dq.convert(g,’K/Jy’);
n     [value=0.106660865, unit=K/Jy]
n   - Tant:='3K’
n   - S:=dq.div(Tant,g)
n   - dq.convert(S,’Jy’)
n     [28.1265297, unit=Jy]
n
