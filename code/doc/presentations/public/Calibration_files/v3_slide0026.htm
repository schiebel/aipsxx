Calibration Components
• Ji contains many components:
• F = ionospheric Faraday rotation
• T = tropospheric effects
• P = parallactic angle
• E = antenna voltage pattern
• D = polarization leakage
• G = electronic gain
• B = bandpass response
• Order of terms follows signal path
• Each term has matrix form of Ji with terms embodying its
particular algebra (on- vs. off-diagonal terms, etc.)
• The full matrix equation (especially after correlation!) is daunting,
but usually only need to consider the terms individually or in
pairs, and rarely in open form (matrix formulation = shorthand)
