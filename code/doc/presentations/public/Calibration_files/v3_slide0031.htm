Polarization Leakage, D
• Polarizer is not ideal, so orthogonal polarizations not perfectly
isolated
• Well-designed feeds have d ~ a few percent or less
• A geometric property of the feed design, so frequency dependent
• For R,L systems, total-intensity imaging affected as ~dQ, dU, so only important
at high dynamic range (because Q,U~d, typically)
• For R,L systems, linear polarization imaging affected as ~dI, so almost always
important
