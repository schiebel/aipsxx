Possible solutions: PSF interpolation
in a image plane deconvolution
• The facet size can be made very small, tending to a pixel. This
makes a dirty image with the ‘w’ induced PHASE term corrected
for. So image is not dephased. But the ‘uv’ projection is not self-
similar for different directions, which implies that the PSF shape
is a function of position.
• The technique involves estimating PSF’s at different positions
and then interpolating in between when deconvolving in the
image plane.
