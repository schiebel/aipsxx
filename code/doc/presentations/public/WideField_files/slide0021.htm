Non-coplanar arrays
Summary
•Simple geometric effect due to non-coplanarity of synthesis arrays
– Apparent shape of array varies across the field of view
•For low frequency imaging with VLA and other non-coplanar arrays, will limit achieved noise level
•Faceted transform algorithm is most widely used algorithm
–AIPS: IMAGR task
–AIPS++: imager (version for parallel computers available), dragon tools
•Processing:
–VLA mostly can be processed on typical personal computer
–A-configuration (74MHz) and E_VLA needs parallelization
