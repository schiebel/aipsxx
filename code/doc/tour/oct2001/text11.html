AIPS++ interfaces

    • Glish provides the prime command line interface
      • Both for user interaction and scripting
    • Standard interfaces for tools and functions are constructed from simple meta-information
      • Both CLIs and GUIs supported
      • Type-specific widgets provide inputs checking, persistence, etc.
      • Specialized widgets simplify retrieval of information from data sources
    • Customized interfaces for some tools are written in Glish
      • e.g. single dish analysis
    • Wizards to guide users through sequence of choices
      • e.g. standard interferometric imaging
    • Vertical integration for complex tools
