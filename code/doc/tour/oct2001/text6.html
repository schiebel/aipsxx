Glish language

    • ~ Public domain IDL
      • C-like syntax instead of FORTRAN-like
      • Allows interactive and non-interactive use
    • Versatile programming language
      • Dynamic typing
      • Whole array processing
      • Regular expressions
      • Heterogeneous records (mixed data types)
      • Inter process communication
      • Event-driven programming
    • Clients can be easily connected to Glish
      • Tk widgets for GUI programming
      • Make client
      • AIPS++ applications
