What is AIPS++ ?

  • Primary goal is to facilitate new science

  • AIPS++…

    • …next-generation data reduction package
    • … targets radio-astronomy, but also has general astronomical tools.
    • … supports new telescopes and observing techniques
    • … supports both near-real-time and off-line reduction on an equal footing
    • … has a command line interpreter and an open tool-kit for flexible scientific use.
    • … is extensible by astronomers, programmers, and observatories.
