year

There is no unique definition, though the colloquial use of the term corresponds to the "tropical year".

See, for example, tropical year, sidereal year, anomalistic year.

Yellow Pages

(yp) Original name for a client-server protocol used by Sun Microsystems to distribute systems configuration data such as user and host names over a network. Since renamed to NIS (Network Information Service) because "Yellow Pages" is a registered trademark of British Telecom. Commands all begin with yp.
