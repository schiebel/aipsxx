
Welcome to the AIPS++ release mirror at site.

This directory contains CDROM images from the latest release of the 
Astronomical Information Processing System (AIPS++).  Contained in this
directory are:
   README.ftp      this file
   README          the README found on Disc 1
   VERSION         one-line text file containing the version & build date
   aips2cd1.iso    the image of Disc 1
   aips2cd2.iso    the image of Disc 2

For more information, see the AIPS++ home page:
http://aips2.nrao.edu

For information on ISO images and creating CDs, we suggest visiting:
http://www.xcdroast.org
