Pre-release test version

Welcome to the AIPS++ Pre-release CDROM. This CDROM contains binary distributions for Linux and Solaris. AIPS++ can be either run directly from this CDROM or installed elsewhere and run from there.

  • To run from the CDROM:
    • Either a_root=/mnt/cdrom ; export a_root
    • Or setenv a_root /mnt/cdrom
    • Either . /mnt/cdrom/aipsinit.sh
    • Or source /mnt/cdrom/aipsinit.csh
    • aips++
  • To install:
    • cd /mnt/cdrom
    • ./INSTALL

As a pre-release tester we need your help to make the upcoming AIPS++ release as bug free as possible. With this in mind, we recommend the following steps:
  • Read our Getting Started in AIPS++. This gives you an overview and gets you started using AIPS++.
  • Register your copy of AIPS++ using the function registerNow()
  • Use AIPS++ for some real work.
  • Review the release notes for this pre-release.
  • If you have questions you wish to send us, please use the ask() function.
  • If you find an obvious bug or just unexpected behavior, please send us a bug report using the bug() function or the bug report web page. You make also consult existing bug reports. If in doubt, send in a bug report.
  • In you send us a bug, please give us accurate information on the relative importance of the bug. Is is a true problem or an enhancement request? If it is a problem, is it a showstopper that prevents you from working? Our priority during the testing period (July - September) is to fix bugs. We will respond to enhancement requests, but probably not until after the September release.

We expect to make the first full release in late September. If you have received a copy of the pre-release CDROM, then you will receive the final version. If not, please email us at aips2-request@nrao.edu with the following information: your name and mailing address.
