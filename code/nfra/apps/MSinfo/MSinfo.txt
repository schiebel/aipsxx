MSinfo - description
====================

The application MSinfo can be used to obtain info on an WSRT MS. It
shows items as begin/end time, frequencies, target RA, DEC, etc.

At present (02NOV2006) the information is obtained from the
NFRA_TMS_PARAMETERS table in the MS. Note that this is a WSRT specific
table that is e.g. not present for a LOFAR MS.

The applications basically uses the class NFRA_MSshow to obtain and
show the info. This class inherits the class NFRA_MS to actually
obtain the info.

Showing the info can be done is several ways:
 - standard - a formatted screen output
 - raw - a kwd=value output
 - readme - special kwd=value format for the readme files in MOM
 - arch - special format for the MySQL Archive table.

Methods used from NFRA_MSshow
-----------------------------

 - setMethod - Set the method to obtain the info. Presently only
               NFRA_MS::NFRA is allowed.
 - getNFRAkwd - Get a kwd=value from the NFRA_TMS_PARAMETERS
 - setInfo - get the info from the MS. The info is loaded into the
             private properties of the class.
 - mergeInfo - For MS's with multiple time slices, info from all
               timeslices will be obtained and merged with already
               existing info in the private proeprties.
 - getError - The class has an error mechanisme. This method gets the
              error from the class.
 - getErrStr - Get the error as a string.
 - isFilled - Flag indicating if any info has already been
              obtained. This is e.g. necesary when merging.
 - show - show the info in the requested mode.
